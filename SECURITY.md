# Security Policy

## Supported Versions

TDSNIO has not yet made a stable release. Until there is a stable release line,
security fixes will be made on the `main` branch.

## Reporting a Vulnerability

Please report known and suspected vulnerabilities privately and responsibly by
using GitHub's "Report a vulnerability" flow for this repository.

**Please do _not_ file a public issue.**

### When to report a vulnerability

- You think you have discovered a potential security vulnerability in TDSNIO.
- You are unsure how a vulnerability affects TDSNIO.

### What happens next?

- I will acknowledge receipt when I can and may ask for additional information.
- I will investigate privately and avoid disclosing details until a fix or
  mitigation is available.
- If the project has published releases by then, I will publish the fix in a
  release and use GitHub security advisories where appropriate.
